NOTFAN or NOT FAN?

$NOTFAN is a meme that represents the dedicated fan base of Notcoin, a gaming project on the Ton blockchain.

The game allows users to mine virtual currency, which can be traded for valuable tokens.

Notcoin is incorporated into the Telegram app and is connected to a token being introduced on The Open Network (TON).
